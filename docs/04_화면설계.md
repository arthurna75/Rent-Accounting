# 화면 설계 문서

## 사이트맵

```
/ (redirect to /dashboard)
├── /auth
│   ├── /login              로그인
│   ├── /register           회원가입
│   └── /onboarding         사업자 정보 등록
└── /dashboard
    ├── /                   대시보드 (KPI + 만기 임박)
    ├── /properties
    │   ├── /               부동산 목록
    │   ├── /new            부동산 등록
    │   └── /[id]
    │       ├── /           부동산 상세
    │       └── /edit       수정
    ├── /contracts
    │   ├── /               계약 목록 (필터: 상태/부동산/기간)
    │   ├── /new            계약 등록
    │   └── /[id]
    │       ├── /           계약 상세 + 납부 이력
    │       └── /edit       수정
    ├── /accounting
    │   ├── /journal
    │   │   ├── /           분개장 (날짜/유형/상태 필터)
    │   │   ├── /new        전표 수동 등록
    │   │   └── /[id]       전표 상세 (승인/역분개)
    │   ├── /chart-of-accounts  계정과목 관리
    │   ├── /depreciation   감가상각 현황 + 일괄 처리
    │   └── /deemed-rental  간주임대료 계산기
    ├── /reports
    │   ├── /balance-sheet      재무상태표
    │   ├── /income-statement   손익계산서
    │   ├── /trial-balance      시산표
    │   ├── /rental-income      임대수익 명세서
    │   └── /deemed-rental      간주임대료 신고 자료
    └── /settings
        ├── /               조직 정보 관리
        ├── /members        사용자/권한 관리
        └── /billing        구독 플랜
```

## 반응형 전략
- `lg:` 이상: 사이드바 + 메인 레이아웃
- `md:`: 사이드바 축약 (아이콘만)
- `sm:` 이하: 햄버거 메뉴 + 드로어

## 컴포넌트 라이브러리
shadcn/ui 기반:
- Table → 목록 화면
- Card → KPI 카드
- Dialog → 등록/수정 모달
- Form + Input + Select → 폼 요소
- Badge → 상태 표시
- Tabs → 상세 화면 탭
- Toast → 성공/에러 알림
- DatePicker → 날짜 선택
